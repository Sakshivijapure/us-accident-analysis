# us-accident-analysis

Project Overview

This project performs an in-depth Exploratory Data Analysis (EDA) on the US Accidents dataset to understand patterns in road accidents across the United States. The analysis focuses on identifying trends related to time, location, weather conditions, and accident severity.

The goal of this project is to extract meaningful insights from large-scale accident data and demonstrate practical data science skills such as data cleaning, visualization, and analysis.
